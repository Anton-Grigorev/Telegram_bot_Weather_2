Телеграм бот Погода_2
Создан для удобства получения информации о погоде и других данных к нужном пользователю городе и профессионального совершенствования в написания исходного кода.
Ключ API для доступа к сервису OpenWeather: https://openweathermap.org/
Необходимо импортировать следующие библиотеки и токены, API:
- import telebot                                  
- import requests                                 
- import json 
- Токен BotFather
- API от OpenWeather
Изменена подача запроса и ответа для пользователя.
Запустить можно в любой среде разработки.
